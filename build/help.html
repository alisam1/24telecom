ADAS

В программе возможны индивидуальные настройки для решения любой задачи по контролю качества вождения. Критерии оценки и коэффициенты задаются для каждого объекта, а модули мониторинга фиксируют максимальные значения уско-рений/торможений.Оценка стиля вождения представляет собой штрафные баллы за нарушения. Мини-мальное количество баллов - высокое качество вождения. Реализована возможность мультипликации - увеличения штрафного балла при учете определенных условий, например: резкое торможение при загруженном автомобиле.
